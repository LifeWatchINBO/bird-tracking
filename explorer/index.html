    Visualizing gull tracking data

    The INBO tracked gulls with the UvA-BITS tracking system. Here, we visualized the maximum distance between the birds and their colony over time and the total distance they travelled during certain time intervals. All code is in Javascript so it runs directly in your browser. The data is available as open data on cartoDB and the visualizations here directly interact with the cartoDB API. If you want to use these visulizations too, checkout the github repository.
